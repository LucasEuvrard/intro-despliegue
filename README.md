# *Forma local de uso*

# Servidor - Backend 

### En la carpeta "backend"

**Instalar** las **dependencias** del proyecto:

```sh
npm install
```

**Comenzar** la **ejecución** del **servidor**:
```sh
npm run dev
```

# Servidor - Frontend

### En la carpeta "frontend"

**Instalar** las **dependencias** del proyecto:

```sh
npm install
```

**Comenzar** la **ejecución** del **servidor**:

```sh
npm run start
```

# Base de datos

### En la carpeta "backend"

Para **iniciar** la **base de datos**:

```sh
docker-compose up -d
```

Para **detener** la **ejecución**:

```sh
docker-compose down
```
